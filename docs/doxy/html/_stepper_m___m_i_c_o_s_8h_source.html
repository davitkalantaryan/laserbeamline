6 // For details see also "Manual MA 1086-A006 D" at www.phytron.de

8 // Every motor has its own MICOS.
9 // Up to 16 MICOS (motors) my be driven through one RS485 interface.
10 // We call the class, that drives the all MICOS at an RS485 interface
11 // a "MICOS_Controller".
12 // The MICOS_Controller will be instantiated once for all MICOS connected through
13 // one RS485 bus. To allow access to the RS485 bus via various RS485-Interfaces,
14 // MICOS_Controller uses a class "RS485_Controller".
15 // Every RS485_Controller is implemented in a file of its own
16 // (as an example see IP_OctalPlus_485.h / .cpp).
17 //
18 // Thus the usual calling sequence to instantiate stepper motors is:
19 //
20 // RS485_Controller * theRS485 = new IP500_Acromag(VME_Addr); //create an RS485_Controller
21 // MICOS_Controller * theMICOS = new MICOS_Controller(theRS485); //create the MICOS_Controller
22 // StepperM_MICOS * stepperMotor0 = new StepperM_MICOS(theMICOS, 0);
23 // StepperM_MICOS * stepperMotor1 = new StepperM_MICOS(theMICOS, 1) ;
