Detailed Description

template<class T>
class messageQ< T >

template class for semaphore-guarded queues.

provides for the safe handling of a message queue in a multi-thread program. A semaphore is used to block a thread from access to the queue while any other thread is just modifying it. An other semaphore is used to block a thread that waits for a message.
