
Following abreviations are used:

BA           -> Bus Address (device number in dasy chain)
ICU          -> Ip Carier Unidig
ICRs232      -> IPCARRIER__RS232
ICX          -> IPCARRIER_X
CN           -> CONTROLLER NUMBER (CONTROLLER_NUMBER)
X.BA         -> X.MOTOR_NUMBER (Bus Address for X direction motor)
Y.BA         -> Y.MOTOR_NUMBER (Bus Address for X direction motor)
Z.BA         -> Z.MOTOR_NUMBER (Bus Address for X direction motor)
Nt           -> Notes
- (in Notes) -> Motor is removed from DOOCS server config file (temporary)
